HomeFieldAdvantage

Home Field Advantage, at a base level, is a website that will allow end-users to leverage machine learning to predict the results of the upcoming Super Bowl within a confidence interval. It is our desire to utilize every aspect of the Computer Science and IT curricula to this point, including, but not limited to:

  • Creating a lightweight Machine Learning model that predicts championship sports game results
  • Storing the website and data in a cloud-based environment
  • Scraping data from an online API into a database
  • Cleaning the data, analyzing it, and feeding it to the ML algorithm
  • Presenting the results on the website
